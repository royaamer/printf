# ft_printf

## Description
This project consists of reimplementing the C standard library function `printf()` under the name `ft_printf`.

## Supported Conversions
- %c : character  
- %s : string  
- %p : pointer  
- %d / %i : signed decimal integer  
- %u : unsigned decimal integer  
- %x : hexadecimal (lowercase)  
- %X : hexadecimal (uppercase)  
- %% : percent sign
